CSS Frameworks Take You Far

I’m a big fan of css frameworks, since I mainly work on backend applications, have a framework to build out the UI for me is great.

I’m okay with the phrase Every website looks like bootstrap, if bootstrap, or any other framework, can get the product out the door and into the hands of users.

I came across a framwork called Bulma which has very simple stying and no javascript. I quickly realased that if you are not building complex or interactive UI’s this framework is great. It really has everything that you need.

In this situation if your UI is very simple and needs minimal interactivty jquery can get you the rest of the way there. nice.

Since this is my css framework of choice now. I decided to make a django app that would allow me to integrate bulma with django widgets.
